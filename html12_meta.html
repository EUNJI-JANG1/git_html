meta태그

        meta태그 : head태그에 필요한 속성 표기
        name속성 : description : 현재 페이지에 대한 설명 기술
                  keywords : 검색엔진이 검색할 수 있도록 검색어 기술
                  author : 홈페이지 작성자 기술
                  refresh : (원하는 간격만큼) 새로고침
        http-equiv 속성 : 페이지 자동 이동
                          refresh 속성값을 쓰고 content에 원하는 지연 시간을 입력;이동할 페이지 기술
